# scroll-hero-animation

This repository contains a small Next.js + Tailwind project demonstrating a scroll-driven hero animation using GSAP.

Run locally:

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`

Build & export static: `npm run build` (the project is a standard Next.js app and can be deployed to Vercel or exported).
